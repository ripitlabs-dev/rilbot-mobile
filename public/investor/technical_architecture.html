Unit Economics: At $49/mo per subscriber with 50 agent runs, blended AI cost is ~$10/subscriber/mo. At 1,000 subscribers that is ~$39,000 gross profit on ~$49,000 revenue — before infrastructure costs of ~$800/mo at that scale.

Scalability Roadmap
Stage Users Infrastructure Est. Monthly Cost
Dev · Now < 100 OrangePi single-board · Cloudflare Tunnel · SQLite · Systemd services ~$50
Seed 100 – 10K AWS EC2 t3.xlarge · RDS PostgreSQL · CloudFront CDN · S3 storage ~$800
Scale 10K – 100K AWS ECS Fargate · Aurora Serverless · ElastiCache Redis · Multi-region failover ~$8,000
Path to Scale: The FastAPI + React stack is cloud-native by design. Migration from SQLite to PostgreSQL requires zero code changes — only a connection string update. Containerization via Docker is complete; ECS deployment is a configuration exercise estimated at 2–3 engineering days.
